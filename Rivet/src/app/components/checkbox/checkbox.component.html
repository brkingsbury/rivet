Checkboxes

Checkboxes allow users to select one or more items from a set, or to turn on an option.

There are 3 styling alternatives for checkboxes. class="small-checkbox", class="read-only" and class="label-above".

There is also 1 attribute for checkboxes, disabled. Readonly is only to be used along with disabled, but disabled can be used by itself.
