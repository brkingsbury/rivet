Drawer

Contains contextual content, similar to a modal but allows some revealing of content behind the panel.

  • drawerOpen <boolean> - sets the drawer state to open or closed
  • toggleDrawer() <function()> - a public function that can be called on the component to toggle the drawer to the opposite state of what it is currently in.
  • openDrawer() <function()> - a public function that can be called on the component to open the drawer
  • closeDrawer() <function()> - a public function that can be called on the component to close the drawer
  • drawerCloseEvent <Output> - emits an event when the drawer is closed
