Expansion Panel

Expandable Line Item

Used in table like lists that have additional information or sub-items to add.

Input bindings:

preventCollapse: boolean - defaulted to false, will lock the expansion state and disable the expansion buttons. (optional)

hideExpansionContent: boolean - defaulted to false, will hide the expand button and the expansion section. Unhide has 625ms delay to allow for the view to update. (optional)

panelExpanded: boolean - defaulted to false, allows access to manual expand the panel. Expansion has 625ms delay to allow for the view to update. (optional)

addButtonCallback: EventEmitter - event to emit on click, will hide the add button if not set. (optional)

deleteButtonCallback: EventEmitter - event to emit on click, while hide the delete button if not set. (optional)

overlayDeleteBtnText: string - text for delete overlay 'delete' button (optional)

overlayCancelBtnText: string - text for delete overlay 'cancel' button (optional)
