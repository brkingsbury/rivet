Expansion Panel

Expandable Line Item (styles to be updated)

Used in table like lists that have additional information or sub-items to add.

Input bindings:

formGroup: FormGroup - must contain a nested form group called 'expandedFields' otherwise will have a null pointer exception. (optional)

hideExpansionContent: boolean - defaulted to false, will hide the expand button and the expansion section. (optional)

addButtonCallback: EventEmitter - event to emit on click, will hide the add button if not set. (optional)

deleteButtonCallback: EventEmitter - event to emit on click, while hide the delete button if not set. (optional)
