Buttons

Primary Buttons

Primary buttons should only be used for high emphasis actions and therefore used sparingly within a page.

Secondary Buttons

Contain actions that are important but not primary. Secondary buttons should not be used with Standard buttons but instead should be used along with Primary or Text buttons.

Text Buttons

Used for less pronounced actions located in Dialogs and Cards and help maintain emphasis on content.

Icon Buttons

Used for one click actions such as Delete and Edit, and do not need a text label

When using icons within buttons, it is required to add aria-label="icon-type" for screen readers to understand the button does.

Button with Icon

Used when text labels are necessary to state what the button does while also including an icon. Buttons that only contain icons need ARIA labels to tell screenreaders what the button is for.

Loading Buttons

Used when an action needs to process, disabling the button and informing the user that it is computing. For more information, see loading

If the button does not contain text saying "loading" and only contains an icon, add aria-label="loading" for screen reader support.
