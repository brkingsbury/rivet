Radio Buttons

Radio buttons are used when there can only be one choice from a list of options

There are 3 styling alternatives for radio-buttons. class="small-radio", class="read-only" and class="label-above".

There is also 1 attribute for radio-buttons, disabled. Readonly is only to be used along with disabled, but disabled can be used by itself.
