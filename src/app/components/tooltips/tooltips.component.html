Tooltips

Tooltip (on mouseover)

A tooltip is used to clarify some detail related to the element it's attached to.

  • 'rvtTooltip' is used to bind a string to the tooltip directive
  • 'rvtTooltipClass' adds a class to the tooltip element

Tooltip (manual trigger)

Manually triggered tooltips can be used to call attention to something that has occured programatically

  • 'rvtTooltip' is used to bind a string to the tooltip directive
  • '[rvtManualTooltipShow]' is a boolean that controls whether a tooltip is visible (set to 'undefined' to allow mouseover functionality)
  • 'rvtTooltipClass' adds a class to the tooltip element
