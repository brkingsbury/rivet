Tooltips

Tooltip (on mouseover)

A tooltip is used to clarify some detail related to the element it's attached to.

  • rvtTooltip <string> - [required] used to bind a string to the tooltip directive
  • rvtTooltipClass <string> - adds a class to the tooltip element.
  • rvtTooltipAbsolutePos <boolean> - Moves the added tooltip element out as a child of the body tag instead of inline, which can be helpful in a context where overflow: hidden is defined. (simply adding the attribute works without passing an argument)
  • rvtManualTooltipShow <boolean> - is a boolean that controls whether a tooltip is visible (set to 'undefined' to allow mouseover functionality)
