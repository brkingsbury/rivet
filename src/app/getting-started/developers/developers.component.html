Importing Rivet into your project

Within the Rivet style module are imports for Angular Material, so there is no need to import Material separately into your project.

Add Rivet to the Dependencies

Within the projects package.json, add "rivet-style": "x.x.x" under dependencies, using the version of Rivet you want to import.
