Rivet Design System

Current version: {{ versions[0].version }} | Roadmap

Rivet is our design system established for use by both developers and designers alike. It allows for developers to easily implement different components without having to worry about styling each implementation. For designers is helps maintain a common style across all components to ensure cohesive design throughout products. Rivet is constantly evolving and improving so be sure to check the changelog for recent changes and the roadmap for future improvements.
